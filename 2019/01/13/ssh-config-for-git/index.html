ssh config for git

If you don’t want execute git command with password, you can do the following:

  1. ssh-keygen -t rsa -b 4096 -C “your_email@example.com“ -f ~/path/key1
  2. ssh-keygen -t rsa -b 4096 -C “your_email@example.com“ -f ~/path/key2
  3. vi ~/.ssh/config Note: you can distinguish different repository with custom Host, then you should replace HostName with Host.
  4. if you don’t edit the config file, then you can use ssh-agent Now you can use git protocol without password. Just test as following:
