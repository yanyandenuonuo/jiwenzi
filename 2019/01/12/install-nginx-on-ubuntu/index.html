install nginx on ubuntu

Reference document:

  • https://nginx.org/en/linux_packages.html

Note:

  • If you login without root, you should add sudo before the command.
  • The ubuntu version is 18.04 lts, if you don’t sure your linux version, you can view the version with lsb_release -a.
    Then you can replace bionic with correct version(also called codename).
  1. touch /etc/apt/source.list.d/nginx.list
  2. vi /etc/apt/source.list.d/nginx.list
  3. [arch=amd64] means just available for amd64
  4. download the nginx key from Nginx Key
  5. apt-key add nginx_signing.key, then you can delete this file
  6. apt update
  7. apt install nginx
  8. service nginx start
  9. the default config directory is /etc/nginx
